Prettynumbers

A Java library that formats numbers to abbreviated form. E.g format the file size 50.000 as 50 Kb

Download this project as a .zip file Download this project as a tar.gz file

Formatting file sizes

PrettyFormat pretty = new PrettyFormat();
pretty.setUnit( ByteUnit.Kb );
pretty.format( new BigDecimal( "2048" ) );  // Returns 2 kb

Formatting file sizes

PrettyFormat pretty = new PrettyFormat();
pretty.setUnit( NumberUnit.MILLION );
pretty.format( new BigDecimal( "1234567" ) ); // 1.23 million

Using preferred unit for bytes (kb in this example)

PrettyFormat pretty = new PrettyFormat();
pretty.setUnit( ByteUnit.getPreferredUnit( new BigDecimal("185000") ) );

Using preferred unit for numbers (millions in this example)

PrettyFormat pretty = new PrettyFormat();
// Sets preferred unit to millions
pretty.setUnit( NumberUnit.getPreferredUnit( new BigDecimal("2500000") ) );

Download with Maven

Add the following repository definition to your pom.xml in repositories section.

<repositories>
    <repository>
        <id>laukvik-releases</id>
        <url>https://github.com/laukvik/laukvik-mvn-repo/raw/master/releases</url>
    </repository>      
</repositories>

And add the dependency configuration as:

<dependency>
    <groupId>org.laukvik</groupId>
    <artifactId>prettynumbers</artifactId>
    <version>1.0</version>
</dependency>
