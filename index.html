Prettynumbers

A Java library that formats numbers to abbreviated form. E.g format the file size 50.000 as 50 Kb

View project onGitHub

Formatting file sizes

PrettyFormat pretty = new PrettyFormat();
pretty.setUnit( ByteUnit.MEGA );
pretty.format( new BigDecimal( "2048" ) );  // Returns 2 k

Formatting file sizes

PrettyFormat pretty = new PrettyFormat();
pretty.setUnit( NumberUnit.MILLION );
pretty.format( new BigDecimal( "1234567" ) ); // 1.23 million

Download with Maven

Add the following repository definition to your pom.xml in repositories section.

<repositories>
    <repository>
        <id>laukvik-releases</id>
        <url>https://github.com/laukvik/laukvik-mvn-repo/raw/master/releases</url>
    </repository>      
</repositories>

And add the dependency configuration as:

<dependency>
    <groupId>org.laukvik</groupId>
    <artifactId>prettynumbers</artifactId>
    <version>1.0</version>
</dependency>
